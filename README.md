# 리눅스 프로세스 관리 명령어 통합 요약

| 명령어 | 기능 | 주요 역할 | 주요 옵션/시그널 |
| :--- | :--- | :--- | :--- |
| **`top`** | **실시간** 시스템 모니터링 | CPU, 메모리, 프로세스 활동을 동적으로 표시 | `q` (종료), `k` (kill), `P` (CPU 정렬) |
| **`ps`** | 프로세스 **스냅샷** 확인 | 특정 시점의 정적인 프로세스 목록 출력 | `aux` (모든 프로세스, 사용자 형식), `-ef` (모든 프로세스, 전체 형식) |
| **`jobs`** | 백그라운드 **작업** 관리 | 현재 셸 세션의 백그라운드/정지된 작업 목록 표시 | `fg %N` (포어그라운드 전환), `bg %N` (백그라운드 재개) |
| **`kill`** | 프로세스에 **시그널** 전송 | 프로세스 종료, 재시작 등 제어 | `15 (SIGTERM)` (정상 종료), `9 (SIGKILL)` (강제 종료) |
