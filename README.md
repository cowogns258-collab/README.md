# 오픈소스 과제 2 20213060 채재훈

| 명령어 | 기능 | 주요 역할 | 주요 옵션/시그널 |
| :--- | :--- | :--- | :--- |
| **top** | 실시간으로 시스템의 현재 활동을 동적으로 보여줌 | CPU, 메모리, 프로세스 활동을 동적으로 표시 | *q* (종료),  *k* (kill),  *P* (CPU 정렬) |
| **ps** | 명령어가 실행된 특정 시점의 정적인 프로세스 정보를 보여줌 | 특정 시점의 정적인 프로세스 목록 출력 | *aux* (모든 프로세스, 사용자 형식), *-ef*(모든 프로세스, 전체 형식) |
| **jobs** | 현재 셸 세션에서 백그라운드 나 정지된 상태로 실행 중인 작업 목록을 보여줌 | 현재 셸 세션의 백그라운드/정지된 작업 목록 표시 | *fg %N* (포어그라운드 전환), *bg %N* (백그라운드 재개) |
| **kill** | 특정 프로세스 또는 작업에 시그널을 보내는 범용 명령어 | 프로세스 종료, 재시작 등 제어 | *15* (SIGTERM) (정상 종료),  *9* (SIGKILL)(강제 종료) |
